# Weather TV :sun_behind_rain_cloud:

<a href="https://weather.gamely.com.br" target="_blank"><img src="https://raw.githubusercontent.com/RodrigoDornelles/RodrigoDornelles/refs/heads/master/media/weather-gly-2504.jpg" align="right" width="50%"></a>

> Cross-platform TV Weather Application / Widget.<br/>(Ginga NCL, Tizen, WebOS)

### Features :tv:

 * **Responsive layout**
 * **Made with [Gly Engine](https://github.com/gamelly/gly-engine)**
 * **Geolocation support** _(only in http mode)_

### Online demo :globe_with_meridians:

 * <http://weather.gamely.com.br>
 * <https://weather720p.gamely.com.br>
 * <https://weather1080p.gamely.com.br>

### How to build :toolbox:

```
git clone https://github.com/gamelly/app-weather-tv
```

```
wget https://get.gamely.com.br/cli.lua
```

```
lua cli.lua build src/main.lua
```
